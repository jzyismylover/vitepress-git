git 可视化 #

在线网址

这个网站提供了 git 命令行操作后的可视化界面，如果我们创建一个分支会出现怎么样的结果，合并分支会出现怎么样的结果，同时它还支持与远程进行交互（应该是内部启动了一个服务器来响应我们的拉取请求），拉取后也有对应的可视化界面，我们可以通过对应的可视化更好地理解理解整个过程（可以结合之前画的图来进行一步步梳理这个过程）。
