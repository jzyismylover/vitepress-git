VitePress #

官方文档

⭐ VitePress是什么。VitePress其实是基于 vite 搭建的一个以 markdown 为中心的静态网站生成器，我们可以使用简单的 markdown 语法开发一个自己的博客，好处在于基本不用去编写额外的代码。

基础配置 #

  • 步骤 1: 创建并进入一个目录
$ mkdir vitepress-starter && cd vitepress-starter
  • 步骤 2: 初始化 npm 配置
$ npm run init
  • 步骤 3: 安装 VitePress到开发环境
$ npm install --save-dev vitepress
  • 步骤 4: 创建挂载目录
$ mkdir docs
  • 步骤 5: 添加一些启动命令

🔑 script 里面其实就是对应的程序脚本，对应 dev 别名就是启动脚本，build 别名就是打包脚本

{
  "scripts": {
    "dev": "vitepress dev docs",
    "build": "vitepress build docs",
    "serve": "vitepress serve docs"
  }
  • 步骤 6: 在本地服务器上启动文档站点
// vite 会启动一个本地服务器来挂载我们的应用
$ npm run dev

    │     │     └── index.md
    │     └── index.md

刚开始的目录结构就是这样子的。

  • config.js -- 里面导出一个 JS 对象，包含了导航栏，侧边栏等非常重要的配置，可以理解为这个就是入口文件了。
  • index.md -- 也就是一开始进去看到的页面
// 比较固定的写法，我们修改一些配置就好了
